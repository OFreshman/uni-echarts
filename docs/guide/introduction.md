# 介绍

Uni ECharts 是适用于 uni-app 的 Apache ECharts 组件，无需繁琐的步骤即可轻松在 uni-app 平台上使用 echarts。

## 🎉 特性

- 🚀 快速上手，与 [Vue ECharts](https://github.com/ecomfe/vue-echarts) 近乎一致的使用体验
- 📱 多端兼容，支持 Web、小程序、APP
- 📦 支持 easycom
- ☕ 支持 TypeScript
- 🍳 支持免费商用
